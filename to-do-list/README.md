# To-Do List App

A simple and clean to-do list web application built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as complete or incomplete
- Remove tasks
- Tasks are saved in browser local storage

## Project Structure
- index.html - Main page structure
- style.css - Styling for the app
- script.js - Task handling and local storage logic
- images/ - Icons and checkbox images

## How to Run
1. Open index.html in your browser, or
2. Use a simple local server such as VS Code Live Server

## Notes
The app stores tasks in the browser so they remain available after refreshing the page.
